About

Hi, I'm Nick. Welcome to my website.

This space serves as my personal playground to practice and experiment with new concepts I encounter in my time spent as a Web-Developer. This page is built from the ground up, with no help from a website builder, using HTML, CSS and Javascript.

I also use the site to showcase other programming projects I've worked on. These include professional projects, where I was part of a larger team, and personal projects I started and finished myself.

Lastly this site is also linked to to my music-production endeavors, under the moniker NHIBIT. Currently this is limited to my Soundcloud profile, but a dedicated NHIBIT website is in the works alongside more original tracks and mixes.

In sync with any new programming projects or music releases, I'll be adding updates to the Recent section of the site.

Thanks for visiting!

Projects Spotlight

Studio Zoomies

Studio Zoomies is a Customer Relationship Management application aimed at photographers. I currently work as a Junior Software Developer on the StudioZoomies Team. Utilizing the JS framework Sveltkit for building the app itself, Firebase for managing app data, and Jira/Bitbucket for version control and project managment, it's been an incredibly rich and insightful look into what it takes to build an app and bring it to market.

Travel App ✈️

Travel-App

This mock travel app was the capstone to my Udacity Front End Web-Developer Nanodegree. It combines all the basic elements required to build a modern web-application. These incude: site construction and styling with html and scss, DOM manipulation using Javascript, calling different APIs and manipulating the returned data, and utilizing build-tools (in this case Webpack) for easier app managment between production and development environments.
